🇦🇿 AZƏRBAYCAN DİLİ:
====================

Nomino - fayl və qovluqların adlandırılması, kopyalanması və idarə edilməsi üçün güçlü alətdir.

📦 QURAŞDIRMA:
--------------
1. Avtomatik quraşdırma üçün: Nomino_1.0.0_x64-setup.exe (tövsiyə edilir)
2. MSI quraşdırıcısı: Nomino_1.0.0_x64_en-US.msi  
3. Birbaşa işə salma: nomino.exe (quraşdırma tələb olunmur)

🔧 MODULLAR:
------------
• Qovluq Adlandırıcı - Excel məlumatlarına əsasən qovluq adlarını dəyişir
• Fayl Adlandırıcı - Excel məlumatlarına əsasən fayl adlarını dəyişir  
• PDF Yaradıcısı - Şəkillərdən PDF faylları yaradır
• Fayl Kopyalama - Faylları Excel siyahısına əsasən kopyalayır
• PDF Tarix Dəyişdiricisi - PDF fayllarında yaranma tarixini dəyişir
• PDF Birləşdirici - Bir neçə PDF faylını birləşdirir
• Excel Fayl Adı Dəyişdirici - Excel məlumatlarına əsasən fayl adlarını dəyişir
• Fayl Sıralayıcı - Faylları qovluqlara simvol uyğunluğuna görə sıralayır

🇷🇺 РУССКИЙ ЯЗЫК:
==================

Nomino - мощный инструмент для переименования, копирования и управления файлами и папками.

📦 УСТАНОВКА:
-------------
1. Автоматическая установка: Nomino_1.0.0_x64-setup.exe (рекомендуется)
2. MSI установщик: Nomino_1.0.0_x64_en-US.msi
3. Прямой запуск: nomino.exe (установка не требуется)

🔧 МОДУЛИ:
----------
• Переименование папок - изменяет имена папок на основе данных Excel
• Переименование файлов - изменяет имена файлов на основе данных Excel
• Создание PDF - создает PDF файлы из изображений
• Копирование файлов - копирует файлы на основе списка Excel
• Изменение даты PDF - изменяет дату создания в PDF файлах
• Объединение PDF - объединяет несколько PDF файлов
• Переименование Excel файлов - изменяет имена файлов на основе данных Excel
• Сортировщик файлов - сортирует файлы по папкам по символьному соответствию

💡 ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ:
-----------------------------
• Поддержка сортировки с азербайджанским алфавитом
• Отслеживание в реальном времени для всех операций
• Подробные журналы и отчеты о результатах
• Современный и удобный интерфейс
